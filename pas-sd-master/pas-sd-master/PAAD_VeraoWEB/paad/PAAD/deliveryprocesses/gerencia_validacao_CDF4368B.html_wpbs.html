Activity: Gerência de Processo Validação
Durante a fase de validação, a disciplina de Gerência de Processo tem com objetivo entregar o Sistema de uma forma satisfatória, verificando sempre a aceitação do cliente como critério de finalização do projeto. Para isso, os trabalhadores cuidam para que o sistema funcione bem no ambiente onde será executado.
